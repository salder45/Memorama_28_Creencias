Memory Game Starter Kit

Author: Eikester
E-Mail: eikester@hotmail.de

Make your own Classic Memory Game with this Starter Kit.

Features:
Flexible Boardsizes, even odd numbers of Pairs (e.g. 5, 7)
There is basically no limit for the number of Pairs
Easily customizable, just add your own Sprites to the GameController Object
Card cover/uncover animation

Kenney.nl for Sounds, Sprites and Font
